Contribution types

There are two different types of contributions you can make, these are:


Topics
Topics are high-level descriptions, like "Racism in the workplace" or "Job opportunities". When adding a topic, we also ask that you include at least one resource, so the topic isn't empty.


Resources
Resources contain the actual information. A resource should be as brief as possible. An example of a resource would be "It's X% more likely for POC to be the subject of Y".
